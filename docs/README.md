# LiteLLM API 文档

欢迎使用 LiteLLM API 文档！本文档采用 GitBook 格式，为您提供详细的 API 参考和使用指南。

## 文档结构

本文档按照以下结构组织：

### 第一级分类
- **对话生成**：包括聊天补全和文本补全功能
- **向量嵌入**：提供文本向量嵌入服务
- **图片生成**：根据文本提示生成图像
- **图片编辑**：编辑和修改现有图像
- **音频转录**：将音频文件转录为文本

### 第二级分类
每个第一级分类下按厂家分类：
- **deepseek**：深度求索
- **zai**：智谱
- **volengine**：字节跳动
- **dashscope**：阿里千问
- **openai**：OpenAI

### 第三级分类
每个厂家下按支持的协议分类：
- **OpenAI**：兼容 OpenAI API 格式
- **Anthropic**：兼容 Anthropic API 格式
- **Gemini**：兼容 Google Gemini API 格式

## 如何使用

1. 选择您需要的功能分类（如对话生成、向量嵌入等）
2. 选择具体的厂家（如 deepseek、openai 等）
3. 选择支持的协议（如 OpenAI、Anthropic 等）
4. 查看详细的 API 文档和使用示例

## API 兼容性

所有 API 均设计为兼容相应协议的标准格式，您可以直接使用现有的客户端库进行调用。
