# Introduction

**NewStream AI Gateway** 

The NewStream AI Gateway  integrates mainstream AI capabilities through a unified API interface, including conversational generation, image generation and processing, audio processing, and vector embeddings. It enables developers to efficiently access multiple AI services, simplifies the complexity of integrating APIs from different providers, and significantly improves development and deployment efficiency.

All APIs are designed with standardized interfaces, allowing flexible adaptation to various business scenarios. The gateway is particularly suitable for edge intelligence and real-time content processing applications.

For access authorization or technical support, please contact the NewStream customer support team: [dev@ngaa.com.cn](mailto:dev@ngaa.com.cn).
